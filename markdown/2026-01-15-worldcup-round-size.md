# 월드컵 강수 선택 (게임 시작 시 설정)

## 목적
- 월드컵 시작 전에 64/32/16/8/4/2강 중 가능한 강수를 유저가 선택한다.
- 아이템 수 기준으로 가능한 강수만 노출되며, 불가능한 강수는 선택할 수 없다.
- 선택 강수보다 아이템이 많으면 플레이 시작 시 랜덤으로 제외하고 진행한다.

## 동작 요약
- 게임 시작 버튼 클릭 → 아레나 화면 모달에서 라운드 드롭다운 선택 → 시작 버튼으로 진행.
- 아이템 수 기준 최대 강수는 자동 계산하며 최대 64강으로 제한한다.
- 선택 강수만큼 랜덤으로 추출하여 월드컵 아레나 진행.

## 관련 변경
- 프론트: 월드컵 시작 화면에 라운드 선택 모달 추가.
- 프론트: 월드컵 아레나에서 선택 강수에 맞게 랜덤 컷 후 진행.
