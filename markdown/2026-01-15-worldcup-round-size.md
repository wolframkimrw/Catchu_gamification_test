# 월드컵 강수 선택 (생성 시 설정)

## 목적
- 월드컵 생성 시 4강/8강/16강/32강 등 진행 강수를 선택할 수 있게 한다.
- 등록된 이미지 수 기준으로 가능한 최대 강수를 안내한다.
- 선택 강수보다 아이템이 많으면 플레이 시 랜덤으로 제외하고 진행한다.

## 동작 요약
- 생성 화면에서 `진행 강수`를 선택하면 `round_size`로 저장된다.
- 백엔드에서 강수는 2의 거듭제곱만 허용하고, 최대값(아이템 수 기준)으로 제한한다.
- 플레이 진입 시 전체 아이템 중 강수만큼 랜덤으로 추출해 진행한다.

## 관련 변경
- DB: `games.Game.worldcup_round_size` 추가.
- API: `GameDetailSerializer`에서 `worldcup_round_size` 제공.
- 프론트: 월드컵 생성 화면에 강수 선택/최대 안내 추가, 드래프트 저장 포함.
- 프론트: 월드컵 아레나에서 강수에 맞게 랜덤 컷 후 진행.
